TaskForge is a front-end web dashboard that helps teams manage tasks, projects, and
workflows in one place. It includes tools for creating tasks, assigning team members,
tracking progress, and organizing work using a Kanban-style interface. The system uses
HTML, CSS, and JavaScript and runs fully in the browser without any backend.

The dashboard supports priority tags, status updates, drag-and-drop task movement, a
modal form for editing tasks, filters, and automatic progress statistics.

---

## Features

### Core Functional Requirements
- Create, edit, and delete tasks
- Organize tasks into three status groups: To do, In progress, Done
- Assign tasks to team members with due dates
- Priority labels: High, Medium, Low
- Search, filter, and sort tasks by status, priority, or due date
- Responsive layout suitable for laptop and mobile
- Progress indicator showing overall project completion
- Upcoming deadlines table
- Local storage support to preserve tasks between page reloads

### Optional Enhancements
- Drag-and-drop task movement for prioritization
- Task reminder or notification integration
- Comment threads or file attachments
- Light and dark themes

---

## Technology Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**  
No frameworks, libraries, or backend services required.

---

## How to Run
1. Download the complete `taskforge.html` file.
2. Open it in any browser (Chrome, Edge, Firefox).
3. Start adding or editing tasks using the **+ New Task** button.
4. Use the filters at the top to refine the task list.
5. Drag a task card across columns to update status.
